# 🤖 AIエージェントへの指示テンプレート

新しいチャットセッションでこのプロジェクトを使う場合の、AIエージェントへの指示テンプレートです。

## 📋 基本テンプレート

```
このプロジェクトは、食事中に取引先の与件を聞いて素早くデモを作成するためのプロジェクトです。

REQUIREMENTS.mdに与件を追加しました。

以下の作業を自動で実行してください:

1. REQUIREMENTS.mdを読んで、必要な型定義をsrc/types/index.tsに追加
2. 定義した型に基づいて、src/data/sampleData.tsに20件程度のダミーデータを生成
3. src/app/page.tsxを編集して、メイン画面のUIを実装

全てのデータはダミーデータで、バックエンドは不要です。
フロントエンドのみで完結するデモを作成してください。
```

## 🎯 詳細テンプレート（必要に応じて）

```
このプロジェクトは、食事中に取引先の与件を聞いて素早くデモを作成するためのNext.jsプロジェクトです。

【プロジェクトの特徴】
- Next.js 16 + TypeScript + Tailwind CSS
- 全てのデータはダミーデータ（バックエンド不要）
- フロントエンドのみで完結
- 高速にデモを作成することを目的とする

【現在の状況】
REQUIREMENTS.mdに以下の与件を追加しました:
[ここに与件の内容を貼り付け]

【依頼内容】
1. REQUIREMENTS.mdを読んで、必要な型定義をsrc/types/index.tsに追加してください
2. 定義した型に基づいて、src/data/sampleData.tsに20件程度のダミーデータを生成してください
   - src/lib/dummyData.tsのユーティリティ関数を使用してください
3. src/app/page.tsxを編集して、メイン画面のUIを実装してください
   - src/components/のコンポーネント（DemoLayout, Card, Button, Table, Badgeなど）を活用してください
   - モダンで美しいUIにしてください

【注意事項】
- バックエンドは一切不要
- 全てダミーデータで動作
- レスポンシブデザインに対応
- 日本語で表示
```

## 🚀 クイックテンプレート（シンプル版）

```
REQUIREMENTS.mdに与件を追加しました。

このプロジェクトは食事中にデモを作るためのものです。
与件を読んで、型定義・ダミーデータ・UIを自動生成してください。

- 型定義: src/types/index.ts
- ダミーデータ: src/data/sampleData.ts
- UI: src/app/page.tsx

全てダミーデータで、バックエンド不要です。
```

## 📝 使い方

### Step 1: REQUIREMENTS.mdに与件を追加

```markdown
【プロジェクト名】
顧客管理システム

【概要】
取引先との顧客情報を管理したい

【主な機能】
1. 顧客一覧表示
2. 顧客詳細表示
3. 顧客検索
```

### Step 2: 新しいチャットを開始

Cursorで新しいチャットを開く。

### Step 3: プロンプトを送信

上記のテンプレートのいずれかを使って、AIエージェントに指示を出す。

### Step 4: 自動生成完了

AIエージェントが自動で以下を生成:
- ✅ 型定義
- ✅ ダミーデータ
- ✅ UI
